# Backend - Django

This directory contains the Django backend for the Cosmo Management SaaS platform.

## Tech Stack
- Python
- Django Framework
- Django REST Framework (for API)

## Structure
This backend will contain:
- Django project configuration
- API endpoints for property management
- Database models for properties, users, and related entities
- Authentication and authorization
- Business logic and services
