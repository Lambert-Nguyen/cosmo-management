# Frontend - Flutter

This directory contains the Flutter frontend for the Cosmo Management SaaS platform.

## Tech Stack
- Dart
- Flutter Framework
- Targeting: Web and iOS platforms

## Structure
This frontend will contain:
- Flutter application code
- UI components for property management
- State management
- API integration with Django backend
- Responsive design for web and mobile

## Setup Instructions
Setup instructions will be added once the Flutter project is initialized.

## Supported Platforms
- Web (primary focus)
- iOS (primary focus)
- Android (future consideration)
