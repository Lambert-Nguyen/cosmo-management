## ğŸ¯ How to Register for AriStay

### Step 1: Get an Invite Code
Before you can register, you need an **invite code** from your administrator. This code determines:
- Your role in the system (Member, Manager, or Admin)
- Which task group you'll be assigned to (Cleaning, Maintenance, Laundry, etc.)
- Your access permissions

**Contact your administrator to get your invite code.**

### Step 2: Access the Registration Page
1. Go to the AriStay login page
2. Click **"Register with invite code"** link
3. You'll be taken to the registration form

### Step 3: Fill Out the Registration Form

#### Required Information
- **Invite Code**: Enter the code provided by your administrator
- **Username**: Choose a unique username (3-150 characters)
- **Email**: Your email address (must be unique)
- **Password**: Create a secure password (minimum 8 characters)
- **Confirm Password**: Re-enter your password

#### Optional Information
- **First Name**: Your first name
- **Last Name**: Your last name

### Step 4: Create Your Account
1. Double-check all your information
2. Click **"Create Account"**
3. If successful, you'll be automatically logged in
4. You'll be redirected to the appropriate dashboard based on your role

## âŒ Common Issues and Solutions

### "Invalid invite code" Error
**What it means:** The code you entered doesn't exist or isn't valid.

**Solutions:**
- Double-check the code for typos
- Make sure the code hasn't expired
- Contact your administrator for a new code
- Ensure the code hasn't already been used (if single-use)

### "Username already exists" Error
**What it means:** Someone else is already using that username.

**Solutions:**
- Try a different username
- Add numbers or variations (e.g., john123, john_smith)
- Use your email address as username

### "Email already exists" Error
**What it means:** That email address is already registered.

**Solutions:**
- Use a different email address
- If you already have an account, try logging in instead
- Contact support if you believe this is an error

### "Passwords do not match" Error
**What it means:** The password and confirm password fields don't match.

**Solutions:**
- Make sure both password fields are identical
- Check for extra spaces or characters
- Re-type both passwords carefully
